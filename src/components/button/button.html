Button

Buttons communicate actions that users can take. Buttons are used in cards, dialogs, forms, and many other components and patterns.
You need to include class btn to use the button component. Buttons are available in two colors:

  • Primary : Black Color
  • Secondary : White Color
Buttons are available in 4 different types:

Basic Buttons

Basic Buttons are available in three shapes and each of them have two more variants. By default the button is box shaped. Use the following classes to change the shapes of buttons.

  • btn-rc
  • btn-pill
And classes solid or outline for the variants.

Button with icon

Button with icons are similar to basic buttons and are available in same shapes and variants. It also have icon along with the text.

Floating Buttons and Link Buttons.

Floating button is fixed with respect to the viewport at the bottom right.
