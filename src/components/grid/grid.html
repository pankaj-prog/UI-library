Grid

Grids provide a common graphic language that makes it easier for both designers and developers to work on a website. It helps users to better understand the information as it's shown in a more structured and logical way
There are two types of grid layouts available and you can create it with the following classes :

  • grid-2c : Two column grid
  • grid-3c : Three column grid

Note: Styling shown in the demo is only for example, grid-2c, grid-3c will only create the structure of grid.
