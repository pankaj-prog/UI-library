Typography

Typography is an essential component of a website's design. Good typography is not only imperative for aesthetic appeal, but also improves site usability when text legibility and readability concepts are applied. Typography is all about proportions and spacing.
You have typography styles for the following:

1. Heading

You can use heading style with custom classes : h1 , h2 , h3 , h4 , h5 and h6 . Or you can also use the heading elements.

This is a Heading - H1
Font-size: 2.0em
This is a Heading - H2
Font-size: 1.5em
This is a Heading - H3
Font-size: 1.25em
This is a Heading - H4
Font-size: 1.17em
This is a Heading - H5
Font-size: 1.0em
This is a Heading - H6
Font-size: 0.83em

2. Text

You can use text styles with custom classes : text-xxxl text-xxl , text-xl , text-lg , text-md ,and text-sm for different sizes of text.

This is triple extra large text

Font-size: 2.0em

This is double extra large text

Font-size: 1.5em

This is extra large text

Font-size: 1.25em

This is large text

Font-size: 1.17em

This is regular sized text

Font-size: 1.0em

This is small text

Font-size: 0.83em

3. Font Weight

You can use different font weights with classes fw-l , fw-r , fw-sb , fw-b ,and fw-xb .

Light 300

Regular 400

Semi Bold 600

Bold 700

Extra Bold 900

4. Text position and Gray text

Use class text-center or text-right to align your text to center or to the right respectively. By default the text is aligned to the left.
You can use the class text-muted to change the color of text to gray which implies a text with lesser importance.
