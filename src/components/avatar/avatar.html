Avatar

The avatar component can be used to represent people or objects.
You need to include avatar class to use avatars. It is available in 4 sizes and you can use the following classes in your avatar for different sizes:

  • avatar-lg : Large Avatar (6rem : 6rem)
  • avatar-md : Medium Avatar (5rem : 5rem)
  • avatar-sm : Small Avatar (4rem : 4rem)
  • avatar-xs : Extra-small Avatar (3rem : 3rem)
Also there are two types of avatar that you can use:

1. Round Avatar

To use the rounded avatars include the class avatar-round along with class avatar and a class for the size of avatar.

2. Squared Avatar

To use the squared avatars include the class avatar-square along with class avatar and a class for the size of avatar.
