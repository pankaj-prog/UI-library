Badge

The Badge Symbol is a small complimentary element that gives a subtle hint for additional information available.
You can use the badge on text, icon or on the avatars with class badge for the badge element and class badge-container to its container element. And depending on the type of container you want a badge on, you can add the following classes to the container.

  • badge-text-container
  • icon-container
  • badge-avatar-container
By default badge color is red. Also the badge size varies with respect to font size of container or the size of avatars.
