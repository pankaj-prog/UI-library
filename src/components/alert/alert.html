Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.
You need to include alert class to use avatars. Alerts are available in 4 different colors depending on the type of alert. You can use the following classes in your alert for different colors:

  • alert-error : Error Alert
  • alert-warning : Warning Alert
  • alert-success : Success Alert
  • alert-info : Info Alert
Also there are two types of alerts that you can use:

1. Basic Alert

Basic alerts can be used using the class alert along with the class for color.
